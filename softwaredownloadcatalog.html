ABOUT THE PROJECT

In my day to day work I use quite a bit of software. The software is delivered in many different forms, from MSI installers, to executables to ISOs. After some time I realised that I had software stewn all over my computer that was not the easiest thing to keep track of. When I realised this I was also experimenting with a javascript UI library called W2UI. A few keystrokes later and I created a software download catalog that uses a JSON file for the entries displayed with links to download the software (well, less of a link and more of a double click on a grid entry record). Using W2UI I was able to create an easily searchable, sortable and somewhat attractive user interface for the software archive I was maintaining. You can download a copy for your own private (or Intranet) use at or have a look at a live demo (fyi - none of the software download links work in the demo).
